# CodeAlpha_Banking_Management_System
This project is a simple Bank Account Management System developed in C programming language. The main aim of this project is to perform basic banking operations in a computerized way. It allows users to create bank accounts and manage account-related activities such as depositing money, withdrawing money, checking balance, and displaying account details.

Each bank account stores information such as account number, account holder name, and account balance. The project uses structures to group all account details together. It also uses file handling to store account records permanently, so the data is not lost when the program is closed.

The program is completely menu-driven, which makes it easy for the user to operate. The user can select any option from the menu according to the required banking operation. Separate functions are created for account creation, deposit, withdrawal, balance enquiry, and displaying all accounts. This makes the code easy to read and maintain.

This project helps beginners understand how transaction-based systems work. It also gives practical knowledge of using structures and files in C. Basic validations are included, such as checking for negative balance, invalid deposit amount, and insufficient balance during withdrawal.

Overall, this Banking System project is a practical mini project that demonstrates how C programming can be used to solve real-world problems.

Key Features : 
1. Create new bank account
2. Deposit money
3. Withdraw money
4. Check account balance
5. Display all account records
6. Permanent storage using files
7. Basic validation for transactions
   
Concepts Used : 
1. C programming
2. Structures
3. File handling
4. Functions
5. Conditional statements
6. Menu-driven programming
7. Basic transaction handling
